# DR Sim Manager

Motion simulation software for racing and flight simulators.

## Download

Download the latest version from the [Releases](https://github.com/DepartedReality/dr-sim-manager-releases/releases/latest) page.

### System Requirements

- Windows 10/11 (64-bit)
- 4 GB RAM minimum
- .NET Desktop Runtime (included in installer)

## Installation

1. Download the latest `DRSimManager-Setup-x.x.x.exe` from [Releases](https://github.com/DepartedReality/dr-sim-manager-releases/releases/latest)
2. Run the installer
3. Launch DR Sim Manager from the Start Menu
4. Enter your license key and email to activate

## Beta Program

Beta builds are available as [pre-releases](https://github.com/DepartedReality/dr-sim-manager-releases/releases). Beta can be installed alongside the stable version.
